# Qalahood Backend — NestJS API

## Стек

| Технология | Назначение  |
| ---------- | ----------- |
| NestJS 10  | Фреймворк   |
| PostgreSQL | База данных |

## Быстрый старт

```bash
# 1. Установить зависимости
npm install

# 2. Скопировать .env
cp .env.example .env
# Заполнить переменные (DB, JWT, AWS S3)

# 3. Запустить в dev режиме
npm run start:dev

# Swagger будет доступен:
# http://localhost:3000/docs

```
